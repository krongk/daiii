关于(About).

这是一个私人书签管理项目，会员可以通过分类和标签的形式保存网络上的所有链接，同时增加了基于Chrome的书签插件，方便一键添加书签内容。

该项目基于rails_apps_composer gem构建，更多Rails_app_composer请参见: https://github.com/RailsApps/rails_apps_composer provided by the RailsApps Project: http://railsapps.github.com/

This application requires:

Ruby version 1.9.2
Rails version 3.2.12
